# Inbox Zero Mail Native SDK 0.1.2

- Fixed Gmail initial synchronization so large mailboxes load reliably instead of stalling or showing only one message.
- Deferred full Gmail message-body downloads until a message is opened, with clear loading and fallback states.
- Added account actions to the File menu, including adding Gmail or Outlook accounts, signing out, and refreshing mail.
- Added a visible Accounts button to make account management easier to discover.

This release is currently available for Apple Silicon macOS as a signed and notarized DMG.
